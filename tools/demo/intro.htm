Welcome

VoltDB is a visionary "NewSQL" platform: the promise to deliver outstanding data access performance for demanding Big Data applications while scaling on commodity hardware.
  • Unlike traditional databases bottlenecked by disk I/O, VoltDB can execute millions of transactions per second!
  • Unlike "NoSQL" solutions, you work with a relational model and avoid data corruption with transactional/concurrency control.

Key/Value Store

VoltDB is far more versatile than what 'NoSQL' systems, but if all you need is a high-speed Key/Value store this sample shows you how.
  • String keys
  • Binary values up to 1MB
Binary storage lets you use any logic you might want on the client side: you could for instance:
  • Serialize client object in binary form
  • Store zipped content/files
  • Store Unicode string data

VoltDB Studio

VoltDB Studio is a perfect companion for your development: test out your SQL, run procedures, monitor your server, etc... You can find Studio in /tools/studio.web

SQLCmd

SQL Command allows you to run SQL statements and procedures in a simple, convenient command-line interface. You can even feed script files to the utility and save the result output to a data file, for your automation needs. You can find Studio in /tools/sqlcmd (then simply run ./sqlcmd)
